¿En qué soy bueno?

Mi trabajo se centra en aprovechar las capacidades de Power BI para crear soluciones de análisis y visualización de datos altamente efectivas y personalizadas para las organizaciones. Esto implica desde el diseño y la transformación de datos hasta la creación de dashboards interactivos y la integración con otras herramientas y plataformas de Microsoft.

ETL

Un rol de suma importancia es la de limpiar y cargar datos de diversas fuentes para que estén listos para el análisis. Esto es fundamental para garantizar que los datos sean precisos, coherentes y accesibles, lo que facilita la toma de decisiones basadas en datos y el descubrimiento de información valiosa.

Analisis de Datos

Como Analista de Datos es importante comprender los datos, establecer objetivos, explorar con visualizaciones iniciales, segmentar datos, usar visualizaciones avanzadas, analizar series temporales y correlaciones, aplicar clustering, validar modelos y finalmente, interpretar resultados para tomar decisiones informadas.

DataViz

Me encargo de la representación gráfica de datos y la información. Mi objetivo principal es hacer que los datos sean más comprensibles y accesibles para las personas a través de diferentes visualizaciones, gráficos, tablas y otros elementos visuales.

Data Challenge - Suscripciones Gym

Este reporte fue el ganador del Data Challenge de Grow up de junio 2023 https://www.growupcr.com/datachallenge, donde se debía trabajar con datos proporcionados y fuentes complementarias para analizar el comportamiento de las suscripciones a un gimnasio. Se aplicó técnicas y funciones de análisis de datos, así como el diseño de visualizaciones creativas y llamativas para presentar los hallazgos encontrados. El objetivo fue desarrollar un informe ejecutivo atractivo utilizando herramientas como Excel, Power Bi y Figma.

Hotel Reservation Data Analysis - The Continental

I present my proposal for the Onyx Data Hotel Reservations challenge. This report is inspired by the Continental Hotel from the Jhon Wuick movie saga, it was very interesting and fun to apply Data Storytelling techniques with data, where you can apply Data cleaning, Dax functions, data design and visualization and many data analysis techniques. 😎 We did something interesting with the data and placed 3 levels of security to access the information: you can be the City Hotel manager where you can only see this data, the Resort manager where you can also only see the Resort data or the General Manager with full access... All this in a public Power Bi link.
